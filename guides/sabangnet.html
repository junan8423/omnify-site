사방넷 API · 연동키 확인

Omnify 재고/WMS 표준은 사방넷 읽기만 지원합니다. 아래 값만 준비하면 됩니다.

Omnify에 필요한 값

1) 사방넷 로그인 ID (샵 ID / 마스터 ID)
2) API 연동키 (API Key / 인증키)
(선택) 물류처 ID · 접속 서브도메인
이지어드민·셀메이트는 Omnify 연동 대상이 아닙니다. 사방넷만 해당합니다.

1) API 서비스 가입 여부 확인

  1. 사방넷에 로그인합니다.
  2. API는 보통 유료 부가서비스(API 서비스)입니다. 미가입이면 「서비스 신청 / 결제」에서 API 서비스를 신청해야 연동키가 발급됩니다.
  3. 이미 다른 툴(풀필먼트·자동수집)과 연동 중이면, 대개 키가 이미 발급되어 있습니다.

2) 연동키 확인 경로

사방넷 로그인 → 마이페이지 → 서비스 관리 → 연동키 관리 (또는 API 인증키 관리)
  1. 사방넷 웹에 로그인한 뒤 우측/상단 마이페이지로 이동합니다.
  2. 서비스 관리 메뉴에서 연동키 관리 또는 API 인증키 관리를 엽니다.
  3. 화면에 표시된 연동키/API Key를 복사합니다.
  4. 로그인에 쓰는 아이디가 샵 ID입니다. (마스터 계정이면 그 ID를 사용)
메뉴명이 「API 인증키 관리」로 표기된 버전도 있습니다. 둘 중 하나를 찾으면 됩니다.

3) 수집 폼에 어떻게 적을까?

  1. 물류에서 사방넷 사용을 선택합니다. (선택하면 API 입력란이 열립니다)
  2. 사방넷 로그인 ID (샵 ID)에 로그인 아이디를 넣습니다.
  3. API 연동키에 「연동키 관리」에서 복사한 키를 붙여넣습니다.
  4. 추가 메모에 담당자·물류처 등을 적어도 됩니다.
  5. 키를 Drive에만 두고 싶다면 폼 메모에 「Drive API 폴더에 업로드함」이라고 적고, 샵 ID만이라도 남겨 주세요.

4) Omnify 사용 범위 (참고)

  • 표준: 재고·출고 현황 조회(읽기)
  • 쓰기(수량 반영·출고 확정): Enterprise 협의·별도 견적 시에만

경로 근거: 사방넷 부가서비스·연동키 관리 안내 및 주요 연동 매뉴얼(마이페이지 → 서비스 관리 → 연동키/API 인증키).
