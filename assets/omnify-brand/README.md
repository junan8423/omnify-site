# Omnify Brand Assets

Omnify 로고 베리에이션 — 다양한 플랫폼·용도별 사용

## 폴더 구조

| 폴더 | 용도 |
|------|------|
| `source/` | 마스터 원본 (1024×1024 PNG) |
| `svg/` | 벡터 원본 (무한 확대) |
| `png/` | 표준 PNG (16~1024px) |
| `webp/` | WebP (32~1024px, 용량 최적화) |
| `favicon/` | favicon.ico + PNG 파비콘 |
| `pwa/` | Apple Touch · Android Chrome · PWA |

## 빠른 참조

| 용도 | 파일 |
|------|------|
| 웹 파비콘 | `favicon/favicon.ico` |
| iOS 홈 화면 | `pwa/apple-touch-icon-180x180.png` |
| Android / PWA | `pwa/android-chrome-192x192.png`, `pwa/android-chrome-512x512.png` |
| SNS 프로필 | `png/omnify-512x512.png` |
| 문서·제안서 | `png/omnify-300x300.png` |
| 앱스토어 | `png/omnify-1024x1024.png` |
| HTML `<link rel="icon">` | `png/omnify-32x32.png` 또는 `favicon/favicon-32x32.png` |

## 생성 정보

- Source: `omnify-icon-nested.png`
- Master: 1024×1024 PNG (LANCZOS 리사이즈)
- Generated by: `scripts/generate_omnify_brand_assets.py`
