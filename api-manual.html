Omnify 데이터 연동을 위한
API Key 발급 매뉴얼

JK글로벌 · Omnify 옴니파이 · Technical Documentation

Omnify는 대표님의 이커머스 매출 데이터를 실시간으로 수집하여 대시보드에 시각화합니다. 이 과정에서 고객님의 정보 보안을 최우선으로 하며, 모든 데이터 접근은 「조회 전용(Read-only)」 권한으로 안전하게 이루어집니다.

다음 가이드를 참고하여 각 플랫폼에서 API 인증 정보를 발급받아 계약서 폼에 입력해 주세요.

스마트스토어 쿠팡 카페24 보안 안내
N

Platform 01

네이버 스마트스토어

Naver Commerce API

스마트스토어센터 > 판매자정보 > API 정보
센터 열기 ↗

발급 절차

  1. 「판매자정보」 메뉴 내 「API 정보」 클릭
  2. 「API 대행사」 항목에서 「직접연동」 선택 또는 Omnify 가이드에 따른 API ID 생성

입력 필요 항목

판매자 ID API ID API Secret Key
CP

Platform 02

쿠팡

Coupang WING API

쿠팡 윙(WING) > 판매자정보 > 업체정보 > API Key 발급
WING 열기 ↗

발급 절차

  1. 쿠팡 윙 접속 후 「판매자 정보」 탭 클릭
  2. 「API Key 발급」 메뉴 진입
  3. 「API Key 생성」 버튼 클릭

입력 필요 항목

Access Key Secret Key Vendor ID (업체코드)
C24

Platform 03

카페24

Cafe24 App / API

카페24 어드민 > 앱 관리 > 내 앱 관리
어드민 열기 ↗

발급 절차

  1. 「앱 관리」 메뉴에서 「내 앱 관리」 접속
  2. 「앱 추가」 버튼을 눌러 Omnify 자동화용 앱 생성 (또는 기존 연동 앱 확인)
  3. 생성된 앱 상세 정보에서 Client ID와 Secret Key 확인

입력 필요 항목

Client ID Client Secret Shop ID (몰 ID)
🔒

Security Notice

보안 안내

권한 범위

Omnify는 위 API 키를 사용하여 주문, 정산, 배송 데이터만을 「조회」하며, 상품 등록이나 삭제 등 「수정」 권한은 사용하지 않습니다.

데이터 관리

발급받으신 API 키는 암호화된 서버에 저장되며, 서비스 해지 시 즉시 파기됩니다.
