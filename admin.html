신규 계약 등록

0. 고객 정보 수집 (최우선)

키 ID·회사명만 넣고 저장하면 바로 수집 링크를 복사해 고객에게 보낼 수 있습니다. API·Drive·사방넷은 고객이 폼에서 직접 작성합니다.

키 ID·회사명으로 테넌트를 저장한 뒤 수집 링크를 발급할 수 있습니다.

제출 전입니다.
네이밍 규칙
· 키 ID * = 등록 시 직접 입력하는 고유값 → 프로젝트/폴더명과 동일하게 사용
· 형식: 영문 소문자 시작 · a-z 0-9 _ - (예: omnify_gotbody, gotbody_2026)
· 연락 이메일은 고유키가 아님 (Gmail·Naver 등 개인메일 허용)
· 영문명은 키 ID 제안용이며, 최종 확정은 키 ID 입력란입니다

필수: 고객사명 · 키 ID · 연락 이메일 · 채널 1개 이상.

키 ID를 입력하면 폴더명·미리보기 경로에 그대로 쓰입니다. 비우면 영문명으로 제안합니다.
